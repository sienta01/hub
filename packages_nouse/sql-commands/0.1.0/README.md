# Espanso SQL Commands
Espanso package with most used commands


## Available matches
| Trigger |     Replace      |
|---------|------------------|
| :CDB:   |  CREATE DATABASE |
| :ADB:   |  ALTER DATABASE  |
| :II:    |  INSERT INTO     |
| :SF:    |  SELECT * FROM   |
| :DF:    |  DELETE FROM     |
| :CTB:   |  CREATE TABLE    |
| :DTB:   |  DROP TABLE      |
| :GB:    |  GROUP BY        |
| :OB:    |  ORDER BY        |
| :WH:    |  WHERE           |


**Note**: This is the first version. More commands will be added in the future based on requests.
